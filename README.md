# About
This is just a silly repo that consists of code generators in different languages to see if a number is even.

# Python Version
```python
x = 10
is_even(x) # This will generate the code

# or
x = 10
func = is_even(x) # This will return a callable than you can call
print(func()) # Prints 'True'
```

# C Version
```c
// compile the code
gcc c_code.c -o is_even

// run the compiled code
.\is_even 10
```
